# Mytragor Simulador

Projeto: simulador Mytragor — código e assets do simulador.

Instruções rápidas para inicializar o repositório Git e enviar ao GitHub (PowerShell):

```powershell
Set-Location -Path "C:\Users\vandr\OneDrive\Área de Trabalho\deploy"
git init
git add -A
git commit -m "Versão inicial do simulador Mytragor"
git branch -M main
git remote add origin https://github.com/mytragor-art/mytragor-simulador.git
git push -u origin main
```

Observações:
- Certifique-se de ter o Git instalado e disponível no PATH antes de rodar os comandos acima.
- Se você usar autenticação por token no GitHub, configure suas credenciais ou use o Git Credential Manager.
